Provides the classes to create a TCP / UDP socket client that communicates using custom serialized Java messages objects.

The Request Socket Client involves one requirement to function: user classes implementing {@link org.codemonkey.swiftsocketclient.ClientMessageToServer} and {@link org.codemonkey.swiftsocketclient.ServerMessageToClient} to register self-encoding/decoding Java message objects with the server.

Note that the client mode (TCP / UDP) doesn't change how the user interacts with the server: the type of transport is managed by the client without breaking API.

@since 1.0
